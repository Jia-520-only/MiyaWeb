## 引言

在这个快速发展的 AI 时代，模型微调已经成为将预训练模型应用到特定领域的重要手段。本文将带你从零开始学习模型微调。

## 环境准备

### Python 环境配置

首先，我们需要准备一个合适的 Python 环境。

```bash
# 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 升级 pip
pip install --upgrade pip
```

### 安装必要的库

```bash
# 安装 PyTorch（根据你的 CUDA 版本选择）
pip install torch torchvision torchaudio

# 安装 Hugging Face Transformers
pip install transformers

# 安装其他必要库
pip install datasets accelerate peft
```

## 数据准备

### 数据集收集

数据是模型微调的基础。你可以从以下渠道获取数据：

1. **公开数据集**：Hugging Face Datasets、Kaggle 等
2. **自有数据**：企业内部的业务数据
3. **合成数据**：通过 GPT 等模型生成

```python
from datasets import load_dataset

# 从 Hugging Face 加载数据集
dataset = load_dataset("imdb")
print(dataset)
```

### 数据清洗与预处理

```python
from transformers import AutoTokenizer

# 加载 tokenizer
tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")

# 数据预处理函数
def preprocess_function(examples):
    return tokenizer(examples["text"], truncation=True, padding="max_length")

# 应用预处理
tokenized_datasets = dataset.map(preprocess_function, batched=True)
```

### 训练集/验证集划分

```python
# 划分数据集
train_test_split = tokenized_datasets["train"].train_test_split(test_size=0.1)

train_dataset = train_test_split["train"]
eval_dataset = train_test_split["test"]
```

## 模型微调

### 选择合适的预训练模型

根据任务类型选择合适的模型：

- **文本分类**：BERT、RoBERTa、DistilBERT
- **序列标注**：BERT-CRF、SpanBERT
- **生成任务**：GPT、BLOOM、LLaMA
- **多模态**：CLIP、BLIP

```python
from transformers import AutoModelForSequenceClassification

# 加载预训练模型
model = AutoModelForSequenceClassification.from_pretrained(
    "bert-base-uncased",
    num_labels=2
)
```

### 配置训练参数

```python
from transformers import TrainingArguments, Trainer

training_args = TrainingArguments(
    output_dir="./results",
    learning_rate=2e-5,
    per_device_train_batch_size=16,
    per_device_eval_batch_size=16,
    num_train_epochs=3,
    weight_decay=0.01,
    evaluation_strategy="epoch",
    save_strategy="epoch",
    logging_dir="./logs",
)
```

### 开始训练

```python
from transformers import Trainer

trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=train_dataset,
    eval_dataset=eval_dataset,
    tokenizer=tokenizer,
)

# 开始训练
trainer.train()
```

## 高级技巧

### 使用 LoRA 进行高效微调

```python
from peft import LoraConfig, get_peft_model

# 配置 LoRA
lora_config = LoraConfig(
    r=8,
    lora_alpha=16,
    target_modules=["query", "value"],
    lora_dropout=0.1,
    bias="none",
    task_type="SEQ_CLS"
)

# 应用 LoRA
model = get_peft_model(model, lora_config)
```

### 学习率调度

```python
from transformers import get_scheduler

optimizer = torch.optim.AdamW(model.parameters(), lr=2e-5)
num_training_steps = len(train_dataset) * training_args.num_train_epochs

lr_scheduler = get_scheduler(
    "cosine",
    optimizer=optimizer,
    num_warmup_steps=int(0.1 * num_training_steps),
    num_training_steps=num_training_steps,
)
```

## 模型评估与部署

### 评估模型

```python
# 评估模型
eval_results = trainer.evaluate()
print(eval_results)
```

### 保存和加载模型

```python
# 保存模型
model.save_pretrained("./my_model")
tokenizer.save_pretrained("./my_model")

# 加载模型
from transformers import pipeline

classifier = pipeline("text-classification", model="./my_model")
result = classifier("This is a test.")
print(result)
```

## 常见问题与解决方案

### 显存不足

- 减小 batch size
- 使用梯度累积
- 使用混合精度训练（FP16）
- 使用模型并行

### 训练不收敛

- 调整学习率
- 检查数据质量
- 增加训练轮数
- 尝试不同的优化器
