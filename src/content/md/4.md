## 前言

STM32 系列单片机以其强大的性能和丰富的外设，成为了嵌入式开发的首选平台之一。本文将从硬件搭建到软件开发，全面讲解 STM32 的开发流程。

## 硬件搭建

### 开发板选择

常用的 STM32 开发板包括：

- **STM32F103C8T6**：入门级，性价比高
- **STM32F407**：性能更强，外设丰富
- **STM32H7**：高端系列，适合复杂应用

### 必备工具

1. **STM32CubeMX**：图形化配置工具
2. **Keil MDK** 或 **STM32CubeIDE**：IDE 开发环境
3. **ST-Link**：调试器/编程器
4. **杜邦线、面包板**：实验辅助工具

### 连接示意图

```text
STM32F103C8T6 开发板连接：
┌─────────────────┐
│   STM32 开发板   │
├─────────────────┤
│  3.3V ── 3.3V   │ 供电
│  GND  ── GND    │ 地线
│  PA9  ── TX     │ 串口发送
│  PA10 ── RX     │ 串口接收
│  SWDIO          │ 调试数据
│  SWCLK          │ 调试时钟
└─────────────────┘
```

## 软件开发环境

### 安装 STM32CubeIDE

1. 下载 STM32CubeIDE（官方免费）
2. 安装 Java 运行环境（JRE 8+）
3. 运行安装程序，选择安装路径

### 创建第一个项目

```c
// main.c
#include "stm32f1xx_hal.h"

int main(void) {
  // HAL 库初始化
  HAL_Init();
  
  // 系统时钟配置
  SystemClock_Config();
  
  // GPIO 初始化
  MX_GPIO_Init();
  
  while (1) {
    // 主循环
    HAL_GPIO_TogglePin(GPIOC, GPIO_PIN_13);
    HAL_Delay(1000);
  }
}
```

  TIM_ClockConfigTypeDef sClockSourceConfig = {0};
  TIM_MasterConfigTypeDef sMasterConfig = {0};

  htim2.Init.Prescaler = 8400 - 1;  // 1kHz
  htim2.Init.CounterMode = TIM_COUNTERMODE_UP;
  htim2.Init.Period = 1000 - 1;     // 1秒
  htim2.Init.ClockDivision = TIM_CLOCKDIVISION_DIV1;

  HAL_TIM_Base_Start_IT(&htim2);
}

## 高级应用

### ADC 模数转换

```c
// ADC 初始化
void MX_ADC1_Init(void) {
  ADC_ChannelConfTypeDef sConfig = {0};
  
  hadc1.Instance = ADC1;
  hadc1.Init.ScanConvMode = ADC_SCAN_DISABLE;
  hadc1.Init.ContinuousConvMode = ENABLE;
  hadc1.Init.DiscontinuousConvMode = DISABLE;
  hadc1.Init.ExternalTrigConv = ADC_SOFTWARE_START;
  hadc1.Init.DataAlign = ADC_DATAALIGN_RIGHT;
  hadc1.Init.NbrOfConversion = 1;
  
  HAL_ADC_Init(&hadc1);
  
  sConfig.Channel = ADC_CHANNEL_0;
  sConfig.Rank = ADC_REGULAR_RANK_1;
  sConfig.SamplingTime = ADC_SAMPLETIME_55CYCLES5;
  HAL_ADC_ConfigChannel(&hadc1, &sConfig);
}

// 读取 ADC 值
uint32_t ADC_Read(void) {
  HAL_ADC_Start(&hadc1);
  HAL_ADC_PollForConversion(&hadc1, HAL_MAX_DELAY);
  return HAL_ADC_GetValue(&hadc1);
}
```

### PWM 脉宽调制

```c
// PWM 输出
void MX_TIM3_PWM_Init(void) {
  TIM_OC_InitTypeDef sConfigOC = {0};
  
  htim3.Instance = TIM3;
  htim3.Init.Prescaler = 84 - 1;      // 1MHz
  htim3.Init.CounterMode = TIM_COUNTERMODE_UP;
  htim3.Init.Period = 1000 - 1;       // 1kHz PWM
  htim3.Init.ClockDivision = TIM_CLOCKDIVISION_DIV1;
  
  HAL_TIM_PWM_Init(&htim3);
  
  sConfigOC.OCMode = TIM_OCMODE_PWM1;
  sConfigOC.Pulse = 500;              // 50% 占空比
  sConfigOC.OCPolarity = TIM_OCPOLARITY_HIGH;
  sConfigOC.OCFastMode = TIM_OCFAST_DISABLE;
  HAL_TIM_PWM_ConfigChannel(&htim3, &sConfigOC, TIM_CHANNEL_1);
  
  HAL_TIM_PWM_Start(&htim3, TIM_CHANNEL_1);
}

// 调整 PWM 占空比
void PWM_SetDuty(uint16_t duty) {
  __HAL_TIM_SET_COMPARE(&htim3, TIM_CHANNEL_1, duty);
}
```

## 调试技巧

### 使用 ST-Link 调试

1. 连接 ST-Link 到开发板
2. 在 STM32CubeIDE 中配置调试器
3. 设置断点，启动调试
4. 查看变量、寄存器状态

### 常见问题排查

**问题：程序不运行**
- 检查电源连接
- 确认 BOOT 引脚配置
- 验证晶振是否正常工作

**问题：串口无输出**
- 检查波特率设置
- 确认 TX/RX 引脚连接
- 验证串口线是否损坏

## 实战项目示例

### 智能温控系统

```c
// 主程序逻辑
void SmartTempControl(void) {
  float temperature = ReadTemperature();
  
  if (temperature > 30.0) {
    PWM_SetDuty(800);  // 降低风扇速度
  } else if (temperature < 25.0) {
    PWM_SetDuty(200);  // 提高风扇速度
  } else {
    PWM_SetDuty(500);  // 正常速度
  }
  
  // 显示温度
  char tempStr[50];
  sprintf(tempStr, "Temp: %.1f°C", temperature);
  OLED_ShowString(0, 0, tempStr);
}
```
